# ToggleCorp Task-Tracker

A simple and fast task tracking application built for the browser. Organize your tasks into projects, track their progress, and manage everything.

## Features

- **Project Management**: Create unlimited projects to organize your work.
- **Kanban Board**: Track tasks across "To Do", "In Progress", and "Done" statuses.
- **Instant Search**: Filter tasks by title or description in real-time.
- **Keyboard Accessible**: Fully navigable without a mouse.

## Getting Started

### Prerequisites
- Node.js 18.x or later
- npm, yarn, pnpm, or bun

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AmritBhusal/tracker.git
   cd tracker
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the app.
